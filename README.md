socket.io sample project
====

とりあえずWebSocketでチャット

参考: [Socket.IO — Chat | Socket.IO](https://socket.io/get-started/chat/#Getting-this-example)

## Requirements
- Node.js (npm)

## Installation
Install dependencies

### Server

```
cd server
npm install
cd ../client
npm install
```

## Server run

```
cd server
node index.js
```

## Client dev-server

サーバーを動かしているターミナルと、クライアント用のターミナルを別で開いておく。

```
npm run serve
```
